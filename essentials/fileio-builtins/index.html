FileIO with builtin functions

Perl provides several builtin functions for dealing with files. However builtins almost never throw exceptions so you need to remember to check for errors and use magic variables like $!. Also the user interface is sometimes not the best. In general you are better off sticking with higher level modules like Path::Tiny or IO::All.

However use of these builtins is fairly common so its going to be worth learning how they work.

open($filehandle, $mode, $filename)

open(my $filehandle, "<", "data.txt")
    or die "Can't open data.txt for reading: $!";

Valid values of $mode

open() can do a lot more and can be used in very complicated ways. See the official docs for open() for all the gory details.

Filehandles

Reading from a file

# Each line of the file becomes an element in the list
my @lines   = <$filehandle>; 

# Reads in the next line of the file
my $line = <$filehandle>;

See also the official docs IO Operators.

print($filehandle, @list)

# Write to a file
print $filehandle "I eat danger for breakfast, "\n";

# Write to stdout
print "I eat danger for breakfast, "\n";
print STDOUT "I eat danger for breakfast, "\n";

Example 1: Slurp the whole file into memory

# Open the file read-only
open(my $filehandle, "<", "data.txt")
    or die "Can't open data.txt for reading: $!";

# Read the whole file into an array
my @lines = <$filehandle>; 

# Print the whole file to STDOUT
print @lines;

close($filehandle);
